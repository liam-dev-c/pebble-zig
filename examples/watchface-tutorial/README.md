# Watchface Tutorial

A Zig port of the [Pebble C Watchface Tutorial](https://github.com/coredevices/c-watchface-tutorial).

Features time, date, weather (via PebbleKit JS + Open-Meteo), battery meter, Bluetooth icon, Clay settings (background/text color, °C/°F, show date), and unobstructed area (Quick View) support.

## Building

```sh
pebble build
```
